# Web Scraping: Brazilian presidents speeches

Web scraping project to collect all brazilian presidents speeches from [Biblioteca da presidÃªncia (ex-presidents)](http://www.biblioteca.presidencia.gov.br/presidencia/ex-presidentes/) and [Discursos do Planalto (actual president)](https://www.gov.br/planalto/pt-br/acompanhe-o-planalto/discursos) for further data analysis.

## Technologies:
* [Node.js](https://nodejs.org/)
* [Puppteer](https://pptr.dev/)

## Pre requisites:
* Node.js 12+ version
* NPM 6+ version

## Usage:
* `git clone` the project
* Install all dependencies using `npm install`
* Run the main project using: `node main.js`
